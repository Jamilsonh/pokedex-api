# POKEDEX API

Site desenvolvido para simular uma pokedex, fazendo conexão com uma API (POKEAPI).

### 🛠️ Tecnologias 

- REACT
- REACT ROUTER DOM
- AXIOS
- API (POKEAPI)

### 🔧 Comandos Necessários

- npm install | or | yarn
- npm install react-router-dom@6 | or | yarn add react-router-dom@6
- npm install axios | or | yarn add axios
